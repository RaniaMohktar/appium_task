# Mobile Calculator Automation with Appium & BrowserStack

This project demonstrates a simple automation test for the **Calculator app** on Android using **Appium** and **BrowserStack**.

---

## Project Overview

- The test opens the **Calculator app** on an Android device.
- It clicks the number `5` (as a simple demo of interacting with the app).
- The test runs on a **real device in the cloud** via BrowserStack.

---

## Technologies Used

- **Java** - Programming language for the test.
- **Appium** - Mobile automation framework.
- **BrowserStack** - Cloud platform to run tests on real devices.
- **TestNG** - Testing framework for structuring the test.
